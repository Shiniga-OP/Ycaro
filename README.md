# Ycaro

## Sobre:
Ycaro é um console projetado para ser durável, de baixo consumo de energia, e de custo acessível para o consumidor final e desenvolvedores de jogos.

## Ycaro v1.0
Este modelo protótipo é um console para jogos 16-bit com cartuchos MicroSD de limite de código binário 4 MBs de Flash.

## Especificações técnicas:
1. Processador RISC-V, 160 MHz de relógio, 32-bit. aproximadamente 40-140 instruções Assembly.
2. Memória RAM de 400 MBs, mas apenas 320 MBs livres para jogos.
3. Memória de armazenamento Flash 3.9 MBs livres, subtraindo o consumo aproximado do sistema operacional/bootloader.
4. Tela 1.47 polegadas de 172x320 pixels.
5. Conector de alimentação e transferência de dados USB-C.
6. Slot de MicroSD.
7. Alimentação 5V.
8. Possuí Wi-Fi.
9. Possuí Bluetooth.
10. Linguagem principal como Assembly RISC-V.
